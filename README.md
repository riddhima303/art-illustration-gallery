## What This Project Does
This website shows:

- **Home:** Main artworks and welcome section  
- **About Us:** Information about the gallery  
- **Current Events:** Latest exhibitions and workshops  
- **Gallery Info:** Opening hours, tickets, and address  
- **Contact:** Social media links and contact information  

You can click the menu links, and it will **scroll smoothly to that section** on the same page.

---

## Features
- **Sticky Navigation Bar:** The menu stays at the top when you scroll  
- **Smooth Scrolling:** Clicking links goes to sections smoothly  
- **Artwork Gallery:** Shows images with nice hover effect  
- **Responsive Design:** Looks good on phone, tablet, and desktop  
- **Events Section:** Displays upcoming events  
- **Footer:** Shows contact info, social links, and policies  

---

## Tech Used
- HTML  
- CSS  
- Font Awesome (for social icons)  
- `.webp` images for fast loading  

No JavaScript is needed for basic functionality.
